\#Laboratorio. Introducción a Git y GitHub



En esta practica de laboratorio con Git y GitHub, se tiene como objetivo 

comprender que es el control de versiones y que importancia tiene, inicializar un repositorio Git local, hacer un seguimiento de cambios con add, commit, push y pull. Además de crear un repositorio remoto de GitHub, conectar un repositorio local con GitHub y finalmente utilizar comandos básicos de Git desde la terminal o consola.
